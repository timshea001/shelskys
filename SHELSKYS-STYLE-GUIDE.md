# Shelsky's Brand Style Guide

## Fonts

### Primary Fonts (Headlines/Display)
- **Mocha Mattari** (`MochaMattariW01-Regular.woff2`) - Primary headline font
- **Futura BT Bold Condensed** (`FuturaBT-BoldCondensed.woff2`) - Display headlines
- **Arvo** (`Arvo.woff2`, `Arvo-Bold.woff2`) - Serif headlines

### Body/UI Font
- **Nunito** - Available in multiple weights:
  - Regular (`Nunito-Regular.woff2`)
  - Medium (`Nunito-Medium.woff2`)
  - SemiBold (`Nunito-SemiBold.woff2`)
  - Bold (`Nunito-Bold.woff2`)
  - ExtraBold (`Nunito-ExtraBold.woff2`)
  - Italic (`Nunito-Italic.woff2`)
  - MediumItalic (`Nunito-MediumItalic.woff2`)

**All font files are located in the current directory**

## Typography Scale

- **Small**: 13px
- **Medium**: 20px
- **Large**: 36px
- **X-Large**: 42px
- **Button Text**: 1.125em (18px base)

---

## Spacing System

The site uses a modular spacing scale based on rem units:
- **20**: 0.44rem (~7px)
- **30**: 0.67rem (~11px)
- **40**: 1rem (16px)
- **50**: 1.5rem (~24px)
- **60**: 2.25rem (~36px)
- **70**: 3.38rem (~54px)
- **80**: 5.06rem (~81px)

---

## Design Elements

### Buttons
- Border radius: 9999px (fully rounded)
- Primary button background: `#e74b2a`
- Primary button text: `#ffffff`
- Padding: 0.667em vertical, 1.333em horizontal

### Layouts
- Default gap for flex/grid: 0.5em
- Block elements use 50% border-radius for circular effects

---

## Notes for Designers

**Primary headline font is Mocha Mattari** - this should be your go-to for ad creative headlines.

**Brand orange (`#e74b2a`)** is the primary call-to-action color and should be used prominently in advertising materials.

All fonts are provided in modern WOFF2 format. If you need other formats (TTF, OTF), you may need to convert these files using online tools like CloudConvert or FontSquirrel.
